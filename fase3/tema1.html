Etiquetas Semánticas


Son etiquetas introducidas para agregar al diseño de sitios WEB, valor semántico y estructural; que quiere decir esto que a simple vista el usuario puede observar las distintas partes de las cuales consta la página.

heater
Etiqueta creada para marcar un grupo de elementos de instrucción o navegación dentro del documento.

hgroup
Se usa para agrupar encabezados.

nav
Se usa para marcar la sección encargada de navegar a través de la WEB.

article
Se usa para agrupar contenido que tendrá sentido fuera de la página actual.

section
Se usa para marcar una sección genérica y agrupa contenido.

aside
Se usa para aislar una parte del contenido, que está relacionado con la página, pero no hace parte de ella.

footer
Se usa para marcar el pie de pagina y puede contener licencia, autor , termino de uso, versión, etc.
